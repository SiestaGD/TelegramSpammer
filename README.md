## 📱 Telegram Spammer



Простая и мощная утилита для автоматической отправки сообщений в Telegram, Discord, WhatsApp и любые другие мессенджеры.



\*\*Версия 1.0\*\* — стабильный релиз с поддержкой русского языка, настройкой через конфиг и сохранением логов.



## 🎯 Описание проекта



Telegram Spammer — это инструмент для эмуляции клавиатуры. Скрипт копирует ваш текст в буфер обмена, вставляет его (Ctrl+V) и нажимает Enter. Всё, что нужно — кликнуть в поле ввода сообщения и запустить скрипт.



Проект создан для образовательных целей и автоматизации рутинных задач.



## ✨ Особенности



\- ⚡ \*\*Простота\*\* — скачал, настроил в `config.ini`, запустил `start.bat`

\- 🎮 \*\*Работает везде\*\* — Telegram, Discord, WhatsApp, любой текстовый редактор

\- 🔧 \*\*Гибкие настройки\*\* — задержка, количество сообщений, задержка перед стартом

\- 📝 \*\*Логирование\*\* — сохраняет все отправленные сообщения с временем (опционально)

\- 💾 \*\*Поддержка русского языка\*\* — корректно отправляет любой текст через буфер обмена

\- 🛑 \*\*Удобная остановка\*\* — нажмите `Ctrl+C` в любой момент


## 2. Установите Python

Скачайте Python с официального сайта.

Важно: при установке отметьте галочку «Add Python to PATH».



3\. Установите зависимости

Откройте командную строку в папке проекта и выполните:



```

pip install -r requirements.txt
```

4. Настройте конфиг

Отредактируйте файл 
```
config.ini
``` 
в любом текстовом редакторе (Блокнот, Notepad++).

## ▶️ Запуск

Способ 1: через start.bat (рекомендуется)

Отредактируйте config.ini



Дважды кликните на start.bat



У вас есть 3 секунды, чтобы переключиться в Telegram (Alt+Tab)



Кликните в поле ввода сообщения



Спам начнётся автоматически

## 🛑 Управление

Остановить спам: нажмите Ctrl+C в окне консоли



Изменить скорость: измените delay в config.ini



Изменить текст: измените text в config.ini и перезапустите скрипт

## ❓ЧАСТЫЕ ОШИБКИ

❌ Русский текст отправляется как иероглифы или восклицательные знаки
Решение:Убедитесь, что установлен пакет pyperclip:
```
pip install pyperclip
```
❌ Сработает ли на Linux/macOS?

Базовый код должен работать, но start.bat только для Windows. Для Linux/macOS нужно запускать через терминал: python3 spammer.py.
