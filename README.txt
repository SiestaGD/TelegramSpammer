TELEGRAM SPAMMER v1.0
============================================

Простая утилита для автоматической отправки сообщений в Telegram и любые мессенджеры.

------------------------------------------------------------
Как это работает
------------------------------------------------------------

Скрипт эмулирует нажатия клавиш:
1. Копирует текст в буфер обмена
2. Вставляет его (Ctrl+V)
3. Нажимает Enter

Всё, что нужно — кликнуть в поле ввода сообщения и запустить скрипт.

------------------------------------------------------------
Системные требования
------------------------------------------------------------

- Windows 10/11
- Python 3.10 или новее
- 100 MB свободного места

------------------------------------------------------------
Установка
------------------------------------------------------------

1. Распакуйте архив в удобную папку

2. Установите Python с официального сайта:
   https://www.python.org/downloads/
   При установке ОБЯЗАТЕЛЬНО отметьте галочку «Add Python to PATH»

3. Откройте командную строку в папке проекта:
   - Нажмите Shift + ПКМ по папке
   - Выберите «Открыть в терминале»

4. Установите зависимости:
   pip install -r requirements.txt

------------------------------------------------------------
Настройка
------------------------------------------------------------

Откройте файл config.ini в блокноте:

[Settings]
; Текст для спама
text = Привет!

; Задержка между сообщениями (секунды)
delay = 0.5

; Задержка перед стартом (сек)
start_delay = 3

; Сколько раз отправить (0 = бесконечно)
max_messages = 0

; Сохранять логи (True/False)
save_logs = False

Примеры:

- Быстрый спам: text = привет, delay = 0.3
- 50 сообщений: max_messages = 50
- Ожидание старта: delay = 5

------------------------------------------------------------
Запуск
------------------------------------------------------------

ПРОСТОЙ СПОСОБ:
1. Отредактируйте config.ini
2. Дважды кликните на start.bat
3. Быстро переключитесь в Telegram (Alt+Tab)
4. Кликните в поле ввода сообщения
5. Спам начнётся автоматически

ЧЕРЕЗ КОМАНДНУЮ СТРОКУ:
python spammer.py

------------------------------------------------------------
Управление
------------------------------------------------------------

- Остановить спам: нажмите Ctrl+C в окне консоли
- Изменить скорость: поменяйте delay в config.ini
- Изменить текст: поменяйте text в config.ini

------------------------------------------------------------
Возможные проблемы
------------------------------------------------------------

ПРОБЛЕМА: Спамит в консоли, а не в Telegram
РЕШЕНИЕ: Кликните мышкой в поле ввода Telegram

ПРОБЛЕМА: Ошибка «No module named 'pyautogui'»
РЕШЕНИЕ: Установите зависимости через pip install -r requirements.txt

ПРОБЛЕМА: Не удаётся остановить спам
РЕШЕНИЕ: Нажмите Ctrl+C или закройте окно консоли
